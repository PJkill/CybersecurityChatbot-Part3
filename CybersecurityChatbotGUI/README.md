## Features
- 💬 Chat – keyword recognition, random responses, memory, sentiment detection
- 📋 Task Assistant – Add, view, complete, delete tasks with MySQL database
- 🎯 Cybersecurity Quiz – 15 questions, immediate feedback, final score
- 🧠 NLP Simulation – Recognises different phrasings (add task, quiz, activity log)
- 📜 Activity Log – Tracks all actions with timestamps

## Technologies Used
- C# / WPF
- MySQL Database
- MySql.Data Connector

## How to Run
1. Install MySQL Server and create the `cybersecurity_chatbot` database.
2. Run the SQL script to create the `tasks` table.
3. Update `TaskManager.cs` with your MySQL password.
4. Open the solution in Visual Studio.
5. Press `F5` to build and run.
